# Cleaning a dataset for a callcenter with Pandas
This project focuses on a comprehensive Data Wrangling and Preprocessing workflow designed to prepare a raw, high-noise dataset for immediate operational use in a Call Center environment. The primary objective is to rectify common data integrity issues and implement logical filters to provide agents with high-fidelity leads, ready for CRM integration.

The case study is modeled after the practical framework provided by Alex The Analyst, emphasizing the transformation of unstructured Excel data into an optimized format.
